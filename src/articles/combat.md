# Combat

1. Add Leader if not already present

2. Draw Tactic Cards

3. Combat Round

   1. Space Battle
   2. Ground Battle
   3. Retreat (Optional)
   4. Next Round

# Resolving an Attack

1. Roll Dice

2. Combat Actions

   - **Play** Tactic Cards
   - **Draw** Tactic Cards

3. Assign Damage

   - **Hit**: matching color
   - **Direct Hit**: any color

4. Block Damage

5. Destroy Units
